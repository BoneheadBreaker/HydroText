# HydroText

HydroText is a very simple text editor made in C#

you can save, open and edit .txt files

its amazing for learning C# or WPF (gui apps)

#

Feel free to contribute!

## Recently Added

* Ability to make text **bold**, *italic*, <ins>underlined</ins> etc

 all accessable through shortcuts (highlight the text then use the shortcut you want) like ctrl+b for bold, ctrl+i for italic, ctrl+u for underlined
 
 > Warning   
 > Saving in .txt format removes all bold, italic and underlined etc text   
 > 
 > There may be an optional work around in the future, but it will probably make other apps import it weird (like instead of importing **bold** it would import \<bold>text here\</bold>)  
 > But of course that would be optional

* New save formats, .hdt .rtf and .html (and .txt is still supported)
> Info
> HTML cannot be imported, so to not lose your work also export an importable file format
* New load formats, .hdt and .rtf (and .txt is still supported)

* Fixed the enter text zone so it no longer says "RichTextBox" by default
